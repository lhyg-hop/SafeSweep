# SafeSweep

SafeSweep 是一个面向 Windows 10 / Windows 11 的系统盘空间治理助手。

它的核心原则不是“尽可能多删”，而是：

- 先解释，再处理
- 默认保守，优先只读扫描
- 高风险内容优先可恢复
- 系统关键路径只做说明和引导，不直接删除

## 适合谁

SafeSweep 主要面向本地个人用户、小微企业办公电脑和需要给客户交付可视化清理工具的场景。

如果你的需求是：

- 想知道到底是谁占满了 C 盘
- 想安全清理临时文件、缓存、回收站
- 想筛出大文件、下载目录、安装包、聊天附件
- 想通过正规入口卸载大软件
- 想保留操作记录和恢复能力

那么 SafeSweep 就是为这类问题设计的。

## 当前功能

`v0.1.1` 已包含这些能力：

- 快速扫描和深度扫描
- C 盘热点占用识别
- 安全清理候选分层展示
- 预演计划
- 隔离区与恢复
- 操作审计记录
- 软件列表与官方卸载入口
- Windows 官方处理入口聚合
- 扫描快照与增长趋势

## 风险分层

SafeSweep 使用 4 层处理模型：

- `L0 只读说明`
  系统管理空间，例如 WinSxS、休眠文件、分页文件、系统还原点。只解释，不直接删除。
- `L1 安全清理`
  已知可重建的缓存、临时文件、回收站等。允许执行，但仍建议先看预演结果。
- `L2 可恢复处理`
  大文件、下载目录、安装包等。优先进入隔离区，便于恢复。
- `L3 引导处理`
  受保护路径、系统关键位置或高风险对象。只给官方入口和处理建议。

## 下载与使用

前往 GitHub Releases 下载：

- `SafeSweep-v0.1.1-win-x64.zip`
- 或直接下载 `SafeSweep.exe`

推荐使用 zip 包：

1. 下载 `SafeSweep-v0.1.1-win-x64.zip`
2. 解压到任意目录
3. 双击 `SafeSweep.exe`

这个版本是 `win-x64` 自包含发布包，目标是“下载后直接运行”，不要求用户单独安装 .NET Runtime。

## 使用建议

首次使用建议按这个顺序：

1. 先点“快速扫描”
2. 看首页四张卡片和候选项说明
3. 对 `L1` 项先做预演
4. 对 `L2` 项确认是否需要恢复能力
5. 最后再执行计划

如果客户机器上没有可用的非系统盘隔离区，SafeSweep 会明确提示：

- 要恢复能力，可能不能最大化释放 C 盘空间
- 要立刻释放空间，可能需要牺牲完整回滚能力

## 设计边界

SafeSweep 明确不会做这些事情：

- 不做注册表玄学清理
- 不直接硬删 `Program Files`
- 不直接处理 `C:\Windows` 关键目录
- 不伪装“系统优化”
- 不偷偷替用户删除高风险内容
